## 🛠️ Initialization Protocol
To execute the **Clonverse Kernel** locally on your machine, follow these steps:

### 1. Prepare Environment
Ensure Python 3.10+ is installed. Then, install the system's UI engine:
```bash
pip install rich
```

### 2. Boot the System
Navigate to your project directory and run the bootloader:
```bash
python clonexxa_terminal.py
```

### 3. Authentication
* **Access Level:** Root
* **Password:** `admin`

## ⌨️ Terminal Directives
Use these commands inside the kernel to control the OS:

| Directive | Action |
| :--- | :--- |
| `open [module]` | Deploys a specific STEM/Medical lab (e.g., `open physics`). |
| `plan` | AI-curated 3-phase curriculum generation. |
| `netstat` | Live network trace to IBM Z mainframe nodes. |
| `status` | System diagnostic telemetry and uptime tracking. |
| `matrix` | Initiates visual data-stream analysis. |
| `hex [file]` | Raw hexadecimal memory architecture inspection. |
| `boot gui` | Switches control to the Graphical User Interface. |
| `exit` | Orderly shutdown of the Clonverse Kernel. |
